Materiales

Motor de corriente continua

Podemos comprar generadores o motores en tiendas electronicas para poder generar fuerza en las aspas del molino de viento.

Tubos de metal o palos de madera

Podemos cortar las aspas en forma aerodinámica para mejorar la eficiencia, considerando reciclar materiales para hacerlas más sostenibles.

Placa de metal o madera para estabilizar la estructura

Debe ser lo suficientemente pesada para mantener el equilibrio, pero también lo suficientemente resistente para soportar las fuerzas del viento.

Batería recargable

Selecciona una batería con la capacidad adecuada según la necesidad de cada motor o generador.

Cables de alta resistencia

Estos nos serviran para conectar el generador, la batería y otros componentes eléctricos.

Taladro y brocas

Estas cosas nos van a servir para perforar agujeros para poner algunos cables de nuestro producto.
